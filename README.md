## Build

1. Create a build directory 
```
mkdir build
```
2. Go to build directory 
```
cd build
```
5. Generate makefiles using cmake
```
cmake ../
```
6. Run makefile
```
make
```
7. Run the executable
```
./raytracer <Model_Filepath>.json
```
